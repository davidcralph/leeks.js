# leeks.js <img src='https://u.derpyenterprises.org/IDGl' height='64' width='64'>
Simple ANSI styling for your terminal. 

## Features
* Ultra lightweight (0 dependencies and under 10kb!)
* [Chalk](https://github.com/chalk/chalk)-like API

## Usage
See ``test.js``
