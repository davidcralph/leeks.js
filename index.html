leeks.js by ohlookitsderpy

leeks.js is a Node.JS package that provides simple ANSI styling for your terminal. Featuring a chalk-like API and support for custom colours (8-Bit and RGB/HEX), it has no dependencies and is under 10kb!

leekslazylogger by eartharoid

leekslazylogger is an easy-to-use logging module for Node.JS with support for colours and styles, custom log types, logging to files, timestamps, and more. It is customisable, lightweight, and made with leeks.js.
