leeks.js by David Ralph

leeks.js is a Node.JS package that provides simple ANSI styling for your terminal. Featuring a Chalk-like API and support for custom colours (4-bit, 8-bit, full RGB, hex and CSS keywords), it has no dependencies and a small file size!

leekslazylogger by eartharoid

leekslazylogger is an easy-to-use logging module for Node.JS with support for colours and styles, custom log types, logging to files, timestamps, and more. It is customisable, lightweight, and made with leeks.js.
